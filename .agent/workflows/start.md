# 세션 시작 워크플로우 (Session Start)

> 새로운 대화가 시작될 때마다 아래 절차를 **반드시** 수행합니다.

## 절차

1. **프로젝트 상태 확인**
   - `.agent/PROJECT_STATUS.md`를 읽는다.
   - **현재 단계(Current Stage)**와 **다음 작업(Next Action Hook)**을 파악한다.

2. **현재 역할 장착**
   - 현재 단계에 해당하는 역할 파일(`.agent/rules/NN_ROLE_TASK.md`)을 읽는다.
   - 해당 페르소나를 장착하고 행동 지침을 숙지한다.

3. **권장 모델 안내**
   - 역할 파일에 명시된 **권장 모델(Recommended Model)**을 사용자에게 안내한다.
   - 현재 사용 중인 모델과 다를 경우, 모델 변경을 제안한다.

4. **사용자에게 상태 보고**
   - 아래 형식으로 현재 상태를 간결하게 보고한다:
   ```
   📋 프로젝트 상태 로딩 완료
   - 현재 단계: [단계명]
   - 역할: [역할명]
   - 권장 모델: [모델명]
   - 다음 작업: [Next Action Hook 요약]
   ```

5. **작업 시작**
   - Next Action Hook에 명시된 지침에 따라 작업을 시작하거나, 사용자의 지시를 기다린다.
